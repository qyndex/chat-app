# Chat Application

Real-time chat application with WebSocket rooms, messaging, and file sharing support.

**Stack:** Vite | **Complexity:** complex
**Tags:** `chat`, `websocket`, `real-time`, `messaging`, `socket.io`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
